# OLA Inmobiliaria

## Estructura
- `index.html`
- `styles.css`
- `script.js`
- `data/properties.js`
- `assets/img/ola-logo.png`

## Publicar en GitHub Pages
1. Crea un repositorio llamado `ola-inmobiliaria`
2. Sube todos estos archivos manteniendo la misma estructura
3. En GitHub: Settings -> Pages
4. En Build and deployment elige `Deploy from a branch`
5. Branch: `main` y folder: `/ (root)`
6. Guarda y espera unos minutos

## Siguiente paso recomendado
Reemplazar las imágenes temporales de propiedades por fotos reales y actualizar enlaces de Instagram/Facebook.
